Pelajari Faktorial

📐

Apa itu Faktorial?

Faktorial dari bilangan bulat non-negatif n, dinotasikan n!, adalah hasil perkalian semua bilangan bulat positif dari 1 hingga n.

n! = 1 × 2 × 3 × … × n

Kasus khusus: 0! = 1 (by definition).

🔢

Rumus Matematis

n! = ∏ k (k = 1 to n)
n! = n × (n−1)!

Contoh nilai:

  • 5! = 5×4×3×2×1 = 120
  • 10! = 3,628,800
  • 20! = 2.43 × 10¹⁸
🔁

Algoritma Iteratif

function faktorialIteratif(n) {
  let hasil = 1;
  for (let i = 2; i <= n; i++) {
    hasil *= i;
  }
  return hasil;
}

Menggunakan loop sederhana. Efisien dalam penggunaan memori — tidak membangun call stack.

🌀

Algoritma Rekursif

function faktorialRekursif(n) {
  if (n === 0 || n === 1) return 1;
  return n * faktorialRekursif(n - 1);
}

Memanggil dirinya sendiri hingga mencapai base case. Elegan secara kode namun menggunakan call stack.

Penerapan Faktorial

  • Kombinatorik — C(n,r) = n! / (r!(n-r)!)
  • Permutasi — P(n,r) = n! / (n-r)!
  • Deret Taylor — e^x = Σ xⁿ/n!
  • Probabilitas — Distribusi Poisson
  • Kriptografi — RSA, key generation
  • Machine Learning — Naive Bayes
